**2.6: sum the digit in an integer
step1:
1. collect integer between 0 and 1000
2. separate the hundreth, tens and unit
3. calculate (hundreth + tens + unit)
4. store result as sum
5. display sum
step2:
1. prompt user to enter an integer between 0 and 1000
2. collect the value 
3. store value as number
4. calculate number (mod) 10
5. store value as unit
6. calculate ((number (mod) 100) / 10)
7. store value as tens
8. calculate number / 100
9. store value as hundreth
10. calculate hundreth + tens + unit
11. store value as sum
12. display the sum to user
