2.32: Negative, positive and zero value (using loop)
step1:
1. collect five numbers from user
2. determine the number of +ve, -ve and zero (0) values
3. print their count

step2:
1. counter = 1
2. positivesCount=0
3. negativesCount=0
4. zeroCount=0
5. do .....:
	a. prompt the user to enter the number
	b. collect and store input as number

	c.	if number > 0, increase positivesCount by 1
		else if number < 0, increase negativesCount by 1
		else, increase zeroCount by 1
	d.	increase counter by 1 
 ..... while (counter <= 5)

6. print the number of [positivesCount, negativesCount, zerosCount]
